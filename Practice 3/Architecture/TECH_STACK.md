# Technology Stack

**Product:** Dice Randomizer
**Quality Mode:** MVP (Production-ready with pragmatic compromises)
**Last Updated:** 2026-04-30

---

## Frontend

- **Language:** Vanilla JavaScript (ES6+)
- **Markup:** HTML5
- **Styles:** CSS3 (анимации, Flexbox)
- **Rationale:** Приложение состоит из одной кнопки и двух кубиков. React и прочие фреймворки — избыточны. Vanilla JS быстрее, проще, не требует сборки.

## Статика / Веб-сервер

- **Сервер:** Nginx 1.25 (alpine)
- **Rationale:** Стандартный production-способ раздачи статических файлов. Лёгкий образ, надёжный, хорошо документирован.

## Инфраструктура

- **Контейнеризация:** Docker
- **Базовый образ:** `nginx:1.25-alpine`
- **Хостинг:** локально (или любой VPS/облако с Docker)
- **Rationale:** Docker обеспечивает воспроизводимость среды. Alpine-образ минимален по размеру.

## Без бэкенда

- Нет сервера приложений
- Нет базы данных
- Нет API
- **Rationale:** Вся логика — клиентская. Хранить нечего, обрабатывать на сервере нечего.

## Зависимости

| Зависимость | Версия | Назначение |
|---|---|---|
| nginx | 1.25-alpine | Веб-сервер в Docker |

Внешних JS-библиотек нет. Звуковой эффект — встроенный аудиофайл (`.mp3` / `.ogg`).

## Компромиссы MVP

- Нет unit-тестов (для POC объём кода слишком мал, чтобы это было оправдано)
- Нет CI/CD pipeline (локальный деплой через `docker build`)
- Нет мониторинга и логирования (нет сервера приложений — нечего мониторить)
- Нет HTTPS (предполагается запуск локально или за reverse proxy)
