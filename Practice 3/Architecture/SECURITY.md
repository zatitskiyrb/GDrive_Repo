# Security Architecture

**Product:** Dice Randomizer
**Last Updated:** 2026-04-30

---

## Общая оценка

Приложение не обрабатывает пользовательские данные, не имеет бэкенда и не хранит ничего постоянно. Поверхность атаки минимальна.

---

## Аутентификация и авторизация

**Отсутствуют.** Приложение публично доступно, пользователей нет, сессий нет.

---

## Защита данных

- Нет персональных данных (PII)
- Нет базы данных
- Нет передачи данных на сервер
- Требования GDPR/HIPAA не применимы

---

## Безопасность фронтенда

| Угроза | Статус | Комментарий |
|---|---|---|
| XSS | Минимальный риск | Нет пользовательского ввода, нет innerHTML с внешними данными |
| Secrets в коде | Нет | API-ключей нет |
| Зависимости с уязвимостями | Нет | Нет npm-зависимостей |
| Небезопасные CDN | Нет | Все ресурсы локальные |

---

## Безопасность инфраструктуры

- Nginx раздаёт только статические файлы (нет PHP, нет CGI)
- Контейнер работает от непривилегированного пользователя (nginx default)
- Нет открытых портов, кроме 80 (или маппинга на хосте)

---

## Чеклист

- [x] Нет секретов в коде
- [x] Нет пользовательского ввода (нет риска инъекций)
- [x] Нет внешних зависимостей с известными уязвимостями
- [x] Образ на основе официального nginx:alpine
