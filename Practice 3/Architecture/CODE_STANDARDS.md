# Code Standards

**Product:** Dice Randomizer
**Stack:** Vanilla JS + HTML + CSS
**Quality Mode:** MVP
**Last Updated:** 2026-04-30

---

## Общие принципы

- **Чистый код:** самодокументирующийся, читаемый, без лишних абстракций
- **Минимализм:** YAGNI — не добавляй то, что не нужно прямо сейчас
- **Единственная ответственность:** каждая функция делает одно дело

---

## JavaScript

### Именование

| Что | Стиль | Пример |
|---|---|---|
| Переменные | camelCase | `dieValue`, `isRolling` |
| Функции | camelCase, глагол | `rollDice()`, `renderDie()` |
| Константы | UPPER_SNAKE_CASE | `ANIMATION_DURATION` |
| DOM-элементы | camelCase + суффикс | `rollButton`, `sumDisplay` |

### Функции

- Максимум **30–40 строк** на функцию
- Максимум **3 параметра** (если больше — передавай объект)
- Одна функция = одно действие

### Структура файла

```
1. Константы (вверху)
2. DOM-ссылки
3. Функции-утилиты (getRandomDie, renderDie, playSound)
4. Главная функция (rollDice)
5. Event listeners (внизу)
```

### Что запрещено

- `var` — использовать только `const` / `let`
- `innerHTML` с динамическими данными — риск XSS (не актуально здесь, но правило остаётся)
- Глобальные переменные без необходимости
- Закомментированный код в финальной версии

---

## HTML

- Семантические теги: `<button>`, `<main>`, `<section>`, `<p>`
- `id` — для JS-ссылок, `class` — для стилей
- Атрибут `lang="ru"` на `<html>`
- Один `<h1>` на страницу

---

## CSS

- Именование классов: kebab-case (`.roll-button`, `.die-face`)
- Анимации — через `@keyframes` + класс `.rolling`
- Нет `!important` без крайней необходимости
- Mobile-first: базовые стили для мобильных, медиазапросы для десктопа

---

## Комментарии

Комментировать **почему**, а не что:

```javascript
// setTimeout нужен, чтобы DOM обновился до снятия анимации
setTimeout(() => renderDie(die1El, value1), ANIMATION_DURATION);
```

Не писать очевидное:
```javascript
// BAD: увеличиваем счётчик на 1
counter++;
```

## Чеклист перед сабмитом на ревью

- [ ] Нет `var`
- [ ] Функции не длиннее 40 строк
- [ ] Имена переменных понятны без контекста
- [ ] Нет закомментированного кода
- [ ] Кнопка блокируется во время анимации
- [ ] Ошибка воспроизведения звука обработана (try/catch или `.catch()`)
- [ ] CSS-анимация снимается после броска
- [ ] Нет секретов и API-ключей
