# Data Model

**Product:** Dice Randomizer
**Database:** Отсутствует
**Last Updated:** 2026-04-30

---

## Постоянное хранилище

**Базы данных нет.** Приложение не сохраняет никаких данных между сессиями.

---

## Состояние в памяти (runtime)

Всё состояние приложения хранится в JavaScript-переменных и живёт только во время текущей сессии браузера.

```javascript
// Состояние приложения
{
  isRolling: boolean,   // идёт ли сейчас анимация броска
  die1: number,         // текущее значение кубика 1 (1–6)
  die2: number,         // текущее значение кубика 2 (1–6)
  sum: number           // сумма die1 + die2 (2–12)
}
```

---

## Допустимые значения

| Поле | Тип | Диапазон | Описание |
|---|---|---|---|
| `isRolling` | boolean | true / false | Блокировка кнопки |
| `die1` | integer | 1–6 | Значение первого кубика |
| `die2` | integer | 1–6 | Значение второго кубика |
| `sum` | integer | 2–12 | Сумма кубиков |
