# Product Vision: Dice Randomizer

**Version:** 1.0
**Date:** 2026-04-30
**Created by:** Product Manager Agent

---

## Executive Summary

Dice Randomizer — это простое веб-приложение для броска двух стандартных кубиков (d6), разработанное для детей. Приложение отображает анимированный бросок двух кубиков, воспроизводит звуковой эффект и показывает сумму выпавших значений. Проект создаётся в образовательных целях: для обучения разработке приложений через Claude Code и агентную систему.

## Problem Statement

Когда дети играют в настольные игры, физические кубики могут теряться или быть недоступны. Нужен простой, весёлый и визуально привлекательный цифровой аналог — прямо в браузере, без установки.

## Target Audience

- **Primary:** Дети (6–12 лет) — простой интерфейс, яркая мультяшная графика, звук
- **Secondary:** Разработчики, изучающие Claude Code и агентные системы

## Goals & Objectives

1. Реализовать бросок двух d6 с отображением результатов
2. Добавить анимацию броска кубиков
3. Добавить звуковой эффект при броске
4. Показывать сумму двух кубиков
5. Оформить в мультяшном стиле, привлекательном для детей
6. Упаковать в Docker для запуска в браузере

## Success Metrics

- ✅ Приложение запускается в Docker и открывается в браузере
- ✅ Два кубика выдают случайные значения (1–6)
- ✅ Анимация броска воспроизводится при нажатии кнопки
- ✅ Звуковой эффект слышен при броске
- ✅ Сумма двух кубиков отображается корректно
- ✅ Интерфейс выглядит мультяшно и привлекательно для детей

## Differentiation

Проект не конкурентный — это учебный инструмент. Ценность в том, что он создаётся end-to-end через агентную систему Claude Code.

## Scope (v1)

**In scope:**
- Два кубика d6
- Кнопка "Бросить кубики"
- Анимация броска
- Звуковой эффект
- Отображение суммы
- Мультяшный дизайн
- Dockerfile

**Out of scope (v1):**
- История бросков
- Счёт/очки
- Разные типы кубиков (d4, d8, d20...)
- Мультиплеер
